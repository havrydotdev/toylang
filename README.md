# Yet Another Toy Language

Activate [devenv](https://devenv.sh/):

```bash
devenv shell
```

To build and run in REPL mode:

```bash
zig build run
```

## Examples

Basic math operations:

```javascript
// 4
2 + 2;

// 15129
123 * 123;

// 4233
77 * 55 + 20 - 22;
```

Booleans:

```javascript
// true
2 != 3;

// false
77 * 55 + 20 - 22 == -1;
```

Ternary operator:

```javascript
// 1
2 + 2 == 4 ? 1 : 0;
```
